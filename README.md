# Clavirum ⌨️

Clavirum is a powerful and lightweight automatic keystroke generator built with Python. Designed for efficiency and versatility, it allows users to automate text entry with precision control across multiple languages.

This was generated with the help of AI

## ✨ Features

* **Automatic Keystroke Generation:** Mimics real keyboard input to type any text automatically into your target application.
* **Advanced Macro Control:** Fully customizable typing speed. Set your desired interval in **milliseconds (ms)** for perfect timing.
* **Multilingual Support:** Native support for **7 different languages**, making it accessible for global users.
* **User-Friendly Interface:** Simple and intuitive setup, allowing you to start your macros in seconds.
* **Lightweight Performance:** Optimized Python code that runs smoothly without consuming heavy system resources.

## 🚀 Installation

Follow these steps to get Clavirum running on your local machine.

### Prerequisites
* Python 3.8 or higher.
* Necessary Python libraries (e.g., `pyautogui`, `keyboard`).

### Setup
1. Clone the repository:
   `git clone https://github.com/Axtarres/Clavirum.git`
2. Navigate to the project directory:
   `cd Clavirum`
3. Install the required dependencies:
   `pip install -r requirements.txt`
   *(Note: If a requirements.txt is not provided, manually install the libraries used in the script, such as `pip install pyautogui keyboard`)*

## 💻 Usage

To launch the application, run the following command in your terminal:
`python Clavirum.py`

**How to use:**
1. Select one of the 7 supported languages upon startup.
2. Enter the text you wish to automate in the input field.
3. Define the delay interval (in milliseconds) between each keystroke.
4. Press the **Start** button and focus on the window where you want the text to appear.
